Class: AssetLoader

AssetLoader(game, json)

Load assets from a json file. https://phaser.io/docs/2.6.2/Phaser.Loader.html There are some Phaser.Loader that take in a callback and/or callbackContext. These are: binary, pack, script, withSyncPoints If you want to use these loaders with callbacks, you'll have to do it from code, not json. You can set the value to either the default datatype or an object with the parameters you want to set.
